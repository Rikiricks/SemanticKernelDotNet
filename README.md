🚀 What I Built
This repository showcases my hands-on implementation of **Semantic Kernel** in .NET Core, covering the five most essential building blocks required to create real-world, production-ready AI applications.
✅ Implemented Core Concepts


**Plugins:**
Built custom C# plugins that expose business logic as AI-callable tools, allowing the model to interact with application functions, services, and workflows.


**Planner:**
Implemented agent-style orchestration where Semantic Kernel can intelligently decide which tools to call, in what order, and how to complete multi-step tasks autonomously.


**Memory/Vector Store:**
Developed a Retrieval-Augmented Generation (RAG) flow using embeddings and vector search, enabling the application to retrieve relevant internal knowledge and produce grounded responses.


**Filters:**
Added middleware-style filters for cross-cutting concerns such as logging, safety checks, validation, interception, and execution control across the AI pipeline.


**Chat History:**
Built multi-turn conversational context handling so the assistant can preserve prior interactions, maintain continuity, and deliver more context-aware responses.
